# Your NetWorth App

## Overview
A social networking platform where users can showcase their professional skills and expertise through topics, with voting and referral systems for monetization. The application operates in English language.

## Authentication
- User registration and login using Internet Identity
- Secure user session management across the entire application
- Internet Identity integration for seamless authentication flow

## User Profiles
- Users can create and edit profiles containing:
  - Personal details
  - Skills and experiences
  - Qualifications and specializations
  - Professional strengths and USPs
- Profile visibility controls for subscribers:
  - Public/private profile settings
  - Topic-specific visibility configuration
  - Granular privacy controls for different profile sections
- Profile visibility to other users based on privacy settings

## Topic System
- Users can create topics related to fields, skills, and interests
- Each topic displays:
  - Topic content and description
  - Real-time upvote/downvote functionality with live vote counts
  - Emoji reactions (heart, topic-specific icons)
  - Associated hashtags extracted from topic content
  - Category classification (field, skill, interest, etc.)
- Users can browse all topics
- Enhanced search functionality:
  - Keyword search across topics
  - Hashtag-based search
  - Category filters (field, skill, interest, etc.)
  - Combined search and filter options

## Leaderboard
- Global ranking system displaying users based on total votes received on their topics
- Real-time vote counting with dynamic leaderboard updates
- Live ranking changes as users interact with topics

## Referral System
- Each user receives a unique referral link with their UID
- Track referrals and calculate earnings for payouts
- Referral earnings tracking and payout structure display

## Payment Integration
- Multiple payment options: Stripe, PayPal, UPI, and cryptocurrency
- Services available for purchase:
  - Vote purchasing (minimum $0.5 per topic)
  - Referral services
  - Subscription fees
- Secure payment processing and transaction records

## Navigation System
- Searchable menu links list positioned at both bottom left and top right of the application
- Quick navigation functionality allowing users to find and access any page
- Comprehensive page linking system for easy site navigation

## Pages Structure

- Features page listing core functionality and payment options
- Referrals page showing earnings and payout information
- Contact page with SECOINFI contact details, map, and social media links
- Blog page for topic discussions and community engagement
- Sitemap with navigation to all main pages and comprehensive page links accessibility
